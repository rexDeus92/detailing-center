# Премиум Детейлинг-Центр — Лендинг (Gloss & Protect)

Премиальный одностраничный сайт (Landing Page) для детейлинг-центра с акцентом на профессиональную полировку и нанесение керамического покрытия на автомобили.

🔗 **Демо-версия (GitHub Pages):** [https://rexdeus92.github.io/detailing-center/](https://rexdeus92.github.io/detailing-center/)

---

## 🌟 Ключевые Особенности

*   **Премиальный Дизайн:** Современная темная цветовая палитра с золотыми/бронзовыми металлическими акцентами, размытием фонов (glassmorphism) и плавными переходами.
*   **Интерактивный Слайдер "До/После":** Удобное сравнение состояния кузова (ЛКП) до полировки и после нанесения керамики. Полная поддержка touch-событий (жестов) для мобильных телефонов и drag-and-drop для десктопов.
*   **Многошаговый Калькулятор:** Опросник из 4 этапов, который собирает информацию о классе автомобиля, состоянии лакокрасочного покрытия, выбранных услугах и позволяет прикрепить фото.
*   **Авто-генерация Сообщений в Мессенджеры:** После заполнения калькулятора пользователь перенаправляется в WhatsApp или Telegram с готовым текстом, содержащим все выбранные опции, для быстрой оценки стоимости.
*   **Ленивая Загрузка Карт (Lazy-load Map):** Яндекс.Карта загружается только при нажатии на кнопку "Открыть карту", что значительно ускоряет начальную загрузку страницы.
*   **Полная Адаптивность:** Лендинг оптимизирован под любые устройства (мобильные телефоны, планшеты, широкоформатные мониторы) по сетке Mobile-First.
*   **SEO-оптимизация:** Семантическая разметка (HTML5), прописанные мета-теги и OpenGraph разметка для красивых превью в соцсетях.

---

## 🛠️ Технологический Стек

*   **Разметка:** HTML5 (Семантическая структура)
*   **Стилизация:** CSS3 (Ванильный CSS, CSS-переменные, Flexbox, Grid Layout, Custom Scrollbar)
*   **Логика:** Vanilla JavaScript (без использования тяжелых сторонних фреймворков и библиотек)
*   **Иконки:** Встроенный SVG (для максимальной скорости загрузки)
*   **Шрифты:** Montserrat & Inter (Google Fonts)

## 🚀 Как запустить локально

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/rexDeus92/detailing-center.git
   ```
2. Перейдите в папку с проектом:
   ```bash
   cd detailing-center
   ```
3. Запустите любой локальный сервер. Например, с помощью Python:
   ```bash
   python -m http.server 8080
   ```
   или с помощью Node.js `http-server`:
   ```bash
   npx http-server -p 8080
   ```
4. Откройте в браузере адрес: `http://localhost:8080`
